## 关于考试
考查方向
0、实验原理：
①根据简单公式（基本都是A=BC，A=B/C这样的简单形式）进行定性判断。
②概念辨析。
③原理图：有点儿印象就能答上。（不要求绘制，都是填空题）
④具体的原理分析：比如PN结的工作原理……
1、关键测量量：
能搞清楚有什么物理意义、为什么要测这个，就没有问题
2、测量曲线：
基本形态、特殊点（峰、谷）物理意义；正负号。
3、思考题：
有点儿印象就能答上。
